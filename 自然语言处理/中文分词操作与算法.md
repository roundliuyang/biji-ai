# 中文分词操作与算法

#### 一、使用结巴分词

1、基础用法

```python
import jieba

text = "迅雷不及掩耳盗铃儿响叮当仁不让世界充满爱意难平均主义之势头不可阻挡"
jieba.add_word('铃儿响叮当')
jieba.del_word("不让")

result = jieba.cut(text, cut_all=True)
print(list(result))
```

2、进阶用法

```python
import jiebaseg_list = jieba.cut("我来到北京清华大学", cut_all=True)  # 全模式
print(/".join(seg_list))
seg_list = jieba.cut("我来到北京清华大学", cut_all=False)  # 精确模式，默认
seg_list = jieba.cut_for_search("小明硕士毕业于中国科学院计算所，后在日本京都大学深造")  # 搜索引擎模式
seg_list = jieba.lcut("小明硕士毕业于中国科学院计算所，后在日本京都大学深造")
```

cut函数返回的是一个生成器（generator），每次迭代返回一个分词结果，而lcut函数返回的是一个列表，其中包含了所有的分词结果。所以cut函数更适合在处理大文本时逐个分词处理，而lcut函数更适合在小文本或者需要一次性获取全部分词结果时使用。

4、文本摘要

```python
from jieba import analyse
source = "此外，公司拟对全资子公司吉林欧亚置业有限公司增资4.3亿元，增资后，吉林欧亚置业注册资本由7000万元增加到5亿元。吉林欧亚置业主要经营范围为房地产开发及百货零售等业务。目前在建吉林欧亚城市商业综合体项目。2013年，实现营业收入0万元，实现净利润-139.13万元。"

for x, w in analyse.textrank(source, topK= 10, withWeight=True):    
    print(x, w)
```


6、并行分词

将文本按行分隔后，每行由一个jieba分词进程处理，之后进行归并处理，输出最终结果。这样可以大大提高分词速度。

```python
jieba.enable_parallel(4)     # 开启并行分词模式，参数为并行进程数
jieba.disable_parallel()     # 关闭并行分词模式
```

#### 三、手搓分词算法

```python
sentence = "迅雷不及掩耳盗铃儿响叮当仁不让世界充满爱意难平均主义之势头不可阻挡响叮当"

# 构造分词词典，字典文件位于：C:\Python310\Lib\site-packages\jieba\dict.txt
with open("./dict.txt", encoding="utf-8") as file:
    line_list = file.readlines()

word_list = []
for line in line_list:
    word_list.append(line.split()[0])

# print(word_list)

# 第一种算法：滚动遍历2个字，3个字，N个字，如果发现一个存在于字典中则输出
# 此算法会进行重复的分词，不会遗漏在字典中的分词
# for i in range(2, 10):
#     for j in range(0, len(sentence)-i):
#         word = sentence[j:j+i]
#         if word in word_list:
#             print(word)

# 第二种算法：只要分过的词，将不再参与滚动，而是从其下一个字开始继续分词
# 此算法不会对分词进行重复，但是会遗漏掉一些词语
i = 0
while i < len(sentence):
    for j in range(2, 10):
        if sentence[i:i+j] in word_list:
            print(sentence[i:i+j])
            i = i + j - 1
    else:
        i += 1

# 上述两种分词算法还未完善两个功能：
# 没有对分词按照原语句进行排序，可读性较差
# 某些未出现在字典中的词也没有出现在分词结果中（遗漏），可通过作业尝试解决上述两个问题
```

#### 四、大模型中的处理

```python
from transformers import AutoTokenizer

tokenizer = AutoTokenizer.from_pretrained("bert-base-cased")
sequence = "Using a Transformer network is simple"
tokens = tokenizer.tokenize(sequence)
print(tokens)    # ['Using', 'a', 'Trans', '##former', 'network', 'is', 'simple']
```

大模型的分词，并不是简单的按照空格分词或者按照中文语义分词，有更多细节需要了解，后续再学习。
